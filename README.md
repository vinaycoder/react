# IndiaRush React

Built with:
* Node.js v8.9
* React v16
* Redux
* Express
* Babel
* WebPack 4
* MongoDB (3.4+)

#Git Checkout

git clone git@bitbucket.org:rahi_jain/irush.git  #Change User Name


# Installation

1. sudo npm install

2. sudo npm run build:watch

3. sudo npm start
